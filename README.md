# jeuSerieux
## CodeBuilder : L'Atelier du Programmeur

Jeu sérieux pédagogique web pour l'initiation à la programmation.  
Les élèves assemblent des blocs de commandes, exécutent le code en temps réel et observent l'état des variables et la console de sortie.

🌐 **Démo en ligne** : [https://zakariaelamrani.github.io/jeuSerieux/](https://zakariaelamrani.github.io/jeuSerieux/)

## Objectifs Pédagogiques
- Comprendre l'affectation et l'utilisation des **variables**
- Maîtriser la structure itérative (boucle `répéter`)
- Appliquer la structure alternative (`si...alors...sinon`)
- Développer la démarche algorithmique : écrire → exécuter → observer → corriger

## Comment Jouer
1. Lisez l'objectif du niveau affiché en haut.
2. Cliquez sur les commandes dans la palette pour les ajouter à votre programme.
3. Utilisez **EXÉCUTER** pour lancer le code, ou **ÉTAPE** pour le mode debug.
4. Observez la **Console** et la zone **Variables** pour comprendre l'exécution.
5. Validez le niveau et passez au suivant.
